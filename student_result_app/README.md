# Student Result App

A simple web app to manage and display student results. You can **add students**, **view pass/fail students**, **see the topper**, **calculate total & average marks**, **sort by marks**, and **search by name**.

---

## Features

- Add student name and marks (0–100)  
- Delete students individually  
- Filter students:
  - All  
  - Passed (marks ≥ 40)  
  - Failed (marks < 40)  
  - Topper (highest marks)  
- Calculate total marks and average marks  
- Sort students by marks (descending)  
- Search students by name (live search)  
- Data saved in **Local Storage**, so refreshing the page keeps your students  

---

## How to Use

1. Open `index.html` in your browser.  
2. Enter student **name** and **marks**, then click **Add Student**.  
3. Use the buttons to filter, sort, calculate totals, or search.
